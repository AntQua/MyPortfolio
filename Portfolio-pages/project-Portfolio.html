Cera's Portfolio Group project

Cera Kocher Portfolio

Cera's Portfolio is a group project (team of 4) for the Brainnest Frontend Development Advanced Industry Training program, between February and May 2023.

Cera's Portfolio is a single page who displays the profile and work of the creative director, designer and photographer Cera Kosher, one of my colleagues in the program.

To explore the project, visit the Cera's Portfolio website.

For a deeper look into the project's codebase, you can check out the GitHub repository.

Cera's Portfolio was built using only HTML, CSS and JavaScript and deployed on GitHub Pages.
