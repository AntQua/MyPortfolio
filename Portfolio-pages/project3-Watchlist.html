Watchlist Solo project @LeWagon

movies list by category

WhatchList is a individual project of my Web Development bootcamp at LeWagon, Lisbon, finished in 8th September 2023.

Watchlist is a web application designed to create movies categories and for each category create a personal movies list.

The user can create its own categries of movies and add its favorite movies to each category.

To explore the project, visit the Watchlist website.

For a deeper look into the project's codebase, you can check out the GitHub repository.

Watchlist was built using the Ruby on Rails framework and deployed on Heroku. The app's images are stored on Cloudinary.
