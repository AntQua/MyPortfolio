SandSpot Group project @LeWagon

book your place in the sand

SandSpot is a group project of my Web Development bootcamp at LeWagon, Lisbon, finished in 8th September 2023. It was a collaborative effort by a team of four dedicated students at the bootcamp.

SandSpot is a web application designed to find beaches all arround the world and book a place in the sand.

The app offers 3 booking options for any beach available.

To explore the project, visit the SandSpot website.

For a deeper look into the project's codebase, you can check out the GitHub repository.

SandSpot was built using the Ruby on Rails framework and deployed on Heroku. The app's design was crafted with Figma.
