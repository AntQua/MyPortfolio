My Library Project

MyLibrary is a single page who allows the user to input its favourit books and keeping track of its read status.

Note that there is no back-end so the books are not stored in a database so they will vanish when the page is reloaded. The goal of this project was just for training.

To explore the project, visit the MyLibrary website.

For a deeper look into the project's codebase, you can check out the GitHub repository.

MyLibrary was built using only HTML, CSS and JavaScript and deployed on GitHub Pages.
